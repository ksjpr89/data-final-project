# 서울시 2024년 자치구별 1인가구 표 확인 메모

서울통계 표 원문 페이지에서 2024년 「1인가구(연령별)」 자치구별 총계를 확인했다. 자료갱신일은 2025년 7월 30일이고, 수록기간은 2010년부터 2024년까지이며, 출처는 서울특별시 및 서울특별시기본통계로 표시된다. 표 단위는 가구다.

표에서 확인된 2024년 서울 전체 1인가구는 **1,660,813가구**이고, 강서구는 **106,748가구**다. 강서구는 같은 표에서 관악구 153,605가구 다음으로 매우 큰 규모의 1인가구 집적을 보인다. 주요 비교값은 송파구 89,720가구, 영등포구 79,698가구, 강남구 79,274가구, 마포구 73,327가구, 동대문구 72,355가구, 광진구 72,933가구 등이다. 서울 25개 자치구의 단순 평균은 서울 전체 1,660,813가구를 25개 구로 나눈 **약 66,433가구**이므로, 강서구 106,748가구는 서울 자치구 평균의 약 **1.61배**다.

이 수치는 마곡 권역의 수요를 단순 역세권 유동인구가 아니라 **강서구 자체가 서울 평균 대비 1인가구 절대 규모가 큰 권역**이라는 배경에서 해석해야 함을 보여준다. 다만 강서구 내부에서는 행정동 기준으로 '마곡동'이 별도 집계되지 않기 때문에, 마곡 법정동·마곡산단·이대서울병원 접근권을 가양1동, 발산1동, 공항동, 방화1동 등으로 근사하는 방식이 필요하다.
