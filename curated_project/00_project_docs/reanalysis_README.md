# 마곡 특수성 재분석 작업 폴더

## 목적

이 폴더는 마곡 입지 분석을 단순 역세권 유동인구 중심이 아니라 **공항·병원·기업·1인가구가 겹치는 특수 생활권**이라는 관점에서 재검증하기 위해 만든 작업 공간이다. 분석의 기본 원칙은 결론을 먼저 정하지 않고, 각 가설별로 확인된 사실과 추가 검증이 필요한 부분을 분리하는 것이다.

## 핵심 가설

| 가설 | 분석 초점 | 현재 상태 |
|---|---|---|
| 공항 인근 수요 | 김포공항 여객 규모, 국내선·국제선 비중, 이동 전후 대기 가능성 | 2016~2025년 김포공항 여객 자료 정제 완료 |
| 병원 관련 수요 | 이대서울병원 규모, 응급의료센터 지정, 보호자·교대근무자 체류 가능성 | 공식 소개 및 응급의료센터 근거 메모 완료 |
| 1인가구 수요 | 강서구와 서울 평균 비교, 마곡 후보 생활권 행정동 비교 | 강서구 행정동별 1인가구 및 서울 자치구 평균 비교 완료 |
| 기업·R&D 수요 | 마곡산업단지 입주기업·근무인원·R&D 인력 | 2025년 실태조사 PDF 원문 및 메모 완료 |

## 다음 분석 방향

다음 단계에서는 현재의 가설별 근거를 실제 후보 입지 또는 시설 프로그램과 연결해야 한다. 특히 병원 축은 응급실 내원자 수와 보호자 체류시간, 공항 축은 새벽·심야 이동 및 항공 종사자 거주지, 기업 축은 시간대별 체류와 출장자 수요, 1인가구 축은 마곡 법정동 또는 블록 단위 주거유형 데이터를 추가로 확보하는 것이 필요하다.
