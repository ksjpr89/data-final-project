# 서울시 1인가구(연령별) 통계 출처 메모

서울 열린데이터광장 「서울시 1인가구(연령별) 통계」 페이지를 확인했다. 이 통계는 인구주택총조사 규칙에 근거한 조사·지정통계이며, 2015년 이후에는 행정자료를 활용한 등록센서스 방식으로 집계된다. 공표범위는 서울시 및 자치구이며, 내용은 연령별 1인가구다. 일반가구를 대상으로 집계하고, 집단가구와 외국인가구는 제외한다. 2016년부터 매년 공표되며, 조사기준 및 자료시점은 매년 11월 1일 0시 기준이다. 서울 열린데이터광장 페이지 기준 데이터 갱신일은 2025년 7월 30일이며 원천은 국가데이터처 인구주택총조사다.

분석상으로는 강서구의 절대 1인가구 수를 서울 전체 또는 자치구 평균과 비교할 때, 단순 주민등록 세대가 아니라 **일반가구 기준의 인구주택총조사 1인가구**라는 점을 명확히 해야 한다. 앞서 확보한 강서구 행정동별 1인가구 파일도 동일 계열의 KOSIS 일반가구 기준 자료이므로, 강서구 내부 행정동 비교와 서울 자치구 비교를 같은 기준으로 연결할 수 있다.
