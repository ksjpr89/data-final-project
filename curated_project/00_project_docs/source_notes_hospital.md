# 병원·응급 수요 관련 1차 공식 근거 메모

## 이대서울병원 공식 병원장 인사말

- URL: https://seoul.eumc.ac.kr/intro/greeting2.do
- 확인일: 2026-05-22
- 핵심 확인 사항: 이대서울병원은 2019년 서울특별시 강서구 마곡지구에 **1,014병상 규모**로 개원했다.
- 핵심 확인 사항: 병원은 기준 병실 3인실, 전 중환자실 1인실로 감염병 대비에 최적화된 구조를 언급한다.
- 핵심 확인 사항: 2023년 산하에 이대대동맥혈관병원과 이대뇌혈관병원을 개원했고, **골든아워를 사수해야 하는 초급성기 환자** 및 **전국 중증응급환자 이송**과 관련해 병원 옥상 헬리포트와 응급 이송 시스템을 강조한다.
- 핵심 확인 사항: 이대서울병원은 지하철 5호선 발산역에 직접 연결되어 있으며 주소는 서울특별시 강서구 공항대로 260이다.
- 핵심 확인 사항: 이대서울병원 공식 산하기관 소개 페이지는 2021년 응급의료기관 평가에서 지역응급의료센터 부문 A등급을 획득했고, 2024년 거점 지역응급의료센터로 지정되었다고 설명한다.
- 핵심 확인 사항: 공식 소개 페이지는 입원환자와 외래환자의 동선 분리, 첫 진료 라운지, 힐링가든, 아트큐브 등 환자와 보호자의 정서적 안정과 대기 경험 관련 시설을 강조한다.

> 공식 페이지 인용: “이대서울병원은 2019년 서울특별시 강서구 마곡지구에 1,014병상 규모로 문을 열었습니다.”

> 공식 페이지 인용: “2023년에는 이대서울병원 산하에 이대대동맥혈관병원, 이대뇌혈관병원을 개원하여 골든아워를 사수해야만 하는 초급성기 환자들의 생명을 지키고 있습니다.”

> 공식 페이지 인용: “이대서울병원은 보건복지부와 국립중앙의료원이 실시한 ‘2021년 응급의료기관 평가’에서 지역응급의료센터 부문 A등급을 획득한데 이어 2024년 거점 지역응급의료센터로 지정돼 응급환자 치료에 만전을 기하고 있습니다.”

## 분석상 함의

마곡의 병원 수요 가설은 단순히 ‘병원이 있다’가 아니라, 2019년 이후 1,014병상급 대학병원이 새롭게 들어오며 의료 종사자, 교대근무자, 환자 보호자, 응급·중증 환자 동행자라는 **비정기·단기·야간 체류 수요**가 발생할 수 있다는 방향으로 정교화해야 한다. 특히 2024년 거점 지역응급의료센터 지정 사실은 응급 보호자·중증환자 동행자·야간 대기 수요 가설을 단순 편의시설 수요보다 더 강한 논리로 만든다. 다만 실제 수요 크기는 병상 수, 응급실 운영 현황, 주변 숙박·휴식 공간 공급, 병원 근무자 규모 데이터를 추가로 비교해 검증해야 한다.
