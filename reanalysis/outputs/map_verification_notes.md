# 후보 편의점 지도·외부 검색 검증 메모

## 세븐일레븐S발산역점 검증

외부 검색과 지도형 정보 페이지를 통해 상위 후보 `세븐일레븐S`가 **세븐일레븐S발산역점(마곡)**으로 확인되었다. 확인된 도로명주소는 **서울특별시 강서구 공항대로 지하267(마곡동, 발산역사내)**이며, 이는 원자료의 `서울특별시 강서구 공항대로 267` 및 좌표가 발산역에 붙어 있다는 분석 결과와 일치한다.

해당 페이지의 주변 점포 목록에는 **GS25마곡힐스테이트점, GS25발산파크점, GS25마곡루체점, CU발산열린점, 이마트24마곡골든점, GS25마곡파크뷰점, 세븐일레븐마곡에스비점**이 모두 약 0.1km권에 표시되어, 기존 점포 단위 반경 분석에서 산출된 높은 편의점 밀집도와도 부합한다. 또한 주변 공공·생활시설로 **발산역 화장실 0km, 하바나빌딩 개방화장실 0.11km, 원당근린공원 0.42km, 등빛도서관 0.59km, 발산1동주민센터 0.75km** 등이 표시되어, 500m 내 청년·직장인 생활지원형 공공시설 공급은 약하고 500m 바깥에 도서관·주민센터가 존재한다는 공급 공백 분석과 방향이 일치한다.

> 검증 출처: 전국 종량제 봉투 판매소 지도 `세븐일레븐S발산역점(마곡)` 페이지, 확인일 2026-05-22. 원천 자체가 공공데이터 기반 2차 서비스이므로 최종 발표에서는 **지도 검색 검증 보조자료**로 표기하고, 핵심 근거는 소상공인 상가업소 원자료와 서울시 공개데이터 분석으로 둔다.

## GS25S9공항시장역점 검증

외부 검색 결과에서 후보 `지에스25S9`는 **GS25 S9공항시장역점**으로 확인된다. 검색 결과에 표시된 주소는 **서울특별시 강서구 공항동 산20-2 / 강서구 방화동로 지하30**이며, 원자료의 `서울특별시 강서구 방화동로 30`과 좌표가 공항시장역 권역에 붙어 있다는 분석 결과와 일치한다. 이 후보는 발산·마곡권 후보와 달리 공항권 이동 수요, 역세권, 공항시장 인접 수요를 설명할 수 있어 **공항권 이동노동자·외국인 생활지원형 공공시설**의 비교 후보로 유지한다.

다만 공항권 후보는 상위 마곡권 후보보다 프로그램 점수와 전체 수요 규모가 낮고, 기존 시설물 데이터의 최근성이 제한되어 있어 최종 1순위보다는 **대안 후보 또는 프로그램 유형 비교 사례**로 제시하는 것이 적절하다.

> 검증 출처: 한차오 지도 검색 결과 및 외부 검색 결과, 확인일 2026-05-22. 최종 제출물에서는 지도 검색은 보조 검증으로 표기한다.

## 네이버 지도 직접 확인 제한 및 대체 검증

네이버 지도 직접 검색 페이지는 현재 작업 환경 정책상 접근이 제한되어 자동 검증을 수행하지 못했다. 따라서 후보 점포의 실재성·주변 맥락은 다음의 대체 근거로 교차 확인했다.

1. **소상공인 상가업소 원자료**에서 편의점 업종, 도로명주소, 경위도 좌표를 추출했다.
2. **서울시 역사마스터 좌표**와 병원·의원 위치자료, 생활이동·인구 패널, 서울시 시설물 정보를 결합해 반경 기반 지표를 계산했다.
3. 외부 검색과 공공데이터 기반 지도형 2차 페이지에서 상위 후보의 점포명, 도로명주소, 주변 편의점 및 공공시설 맥락을 확인했다.

추가로 사용자가 직접 네이버 지도에서 확인할 경우 우선 검색어는 `세븐일레븐S발산역점`, `GS25발산파크점`, `GS25마곡힐스테이트점`, `GS25마곡루체점`, `이마트24마곡골든점`, `GS25S9공항시장역점` 순으로 확인하면 된다.

### 공항시장역 후보의 지도형 페이지 추가 확인

한차오 지도형 페이지에서도 후보 점포명이 **GS25 S9공항시장역점**으로 표시되며, 주소는 **서울특별시 강서구 공항동 산20-2 / 강서구 방화동로 지하30**으로 확인되었다. 이 결과는 원자료 `방화동로 30` 후보가 단순 도로변 점포가 아니라 **공항시장역 지하 역세권 점포**라는 점을 보강한다. 따라서 이 후보는 ‘공항권·역세권 이동수요 대응’ 프로그램의 비교 후보로 유지하되, 최종 1순위는 데이터 점수와 공급 공백이 더 강한 발산역·마곡 초입권 후보와 비교해 결정한다.
