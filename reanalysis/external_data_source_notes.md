재분석에서는 먼저 기존 프로젝트에 좌표 파일이 있는지 확인하고, 없을 경우 이 원천의 CSV 또는 API를 통해 지하철역 좌표를 보강한다.


## 서울시 시설물 정보

확인일: **2026-05-22**  
출처: 서울 열린데이터광장, `서울시 시설물 정보`  
URL: https://data.seoul.go.kr/dataList/OA-21698/S/1/datasetView.do

서울 열린데이터광장 설명에 따르면 이 데이터는 서울특별시에 위치한 조사대상 시설물 정보를 제공하며, 다중이용시설, 병의원, 구청, 보건소, 보건지소, 공원, 하천 등이 포함된다. 미리보기 컬럼에는 **시설명, 위도, 경도, 시설용도분류, 소재지 도로명주소, 소재지 지번주소, 데이터 기준일자**가 포함되어 있어 후보 편의점 주변의 기존 공공·생활SOC 공급 여부를 교차 검증하는 보조 자료로 사용할 수 있다.

다만 데이터 갱신일은 **2023-04-21**로 최신성 한계가 있으므로, 최종 보고서에는 현재 공급 현황을 확정하는 자료가 아니라 **공급 검증 보조자료**로 표기한다. Tableau에서는 후보 편의점 좌표와 함께 점 레이어로 올려 반경 300m·500m 내 시설 수를 확인하는 용도로 사용한다.

## 서울시 가구원수별 가구수 동별 통계

확인일: **2026-05-22**  
출처 후보: 서울 열린데이터광장, `서울시 가구원수별 가구수(동별) 통계`  
URL: https://data.seoul.go.kr/dataList/10597/S/2/datasetView.do

마곡의 1인 가구 특수성을 서울 전체 또는 서울 행정동 평균과 비교하기 위해 해당 통계를 확인했다. 검색 결과상 이 통계는 행정동 단위 가구원수별 가구수를 제공하는 자료로, 1인 가구 비중 비교에 적합하다. 다만 브라우저 렌더링에서는 페이지 본문과 다운로드 버튼이 빈 화면으로 표시되어 직접 다운로드를 완료하지 못했다. 다음 분석에서는 공개 API, 대체 통계 페이지, 또는 기존 보유 강서구 패널과 서울 자치구 평균 자료를 결합하는 방식으로 서울 평균 대비 비교를 보완한다.

## 2026-05-22 추가: 대체 공개데이터 경로 접근 제한

마곡 특수성 분석에서 서울 전체·자치구 평균 대비 1인 가구 비중을 비교하기 위해 추가 외부 공개데이터 경로를 탐색했다. `gimi9.com`의 서울 1인 가구 관련 데이터셋 페이지는 브라우저 언어 설정 제한으로 본문 및 다운로드 링크를 확인하지 못했다. 따라서 해당 경로는 현재 분석 근거로 사용하지 않고, 보유 강서구 행정동 패널 및 서울 열린데이터광장·KOSIS 등 원천 통계 확보 가능성을 계속 확인한다.

## 2026-05-22 추가: 서울 데이터 허브 직접 조회 페이지

서울 행정동별 가구원수 통계의 직접 조회 페이지(`https://data.seoul.go.kr/bsp/wgs/dataView/data300View/464.do`)를 확인했다. 검색 결과에는 CSV 다운로드가 가능한 것으로 표시되었으나, 브라우저 렌더링에서는 본문·버튼·스크린샷이 로딩되지 않아 다운로드 링크를 확정하지 못했다. 이 경로 역시 현재 분석에 수치 근거로 직접 사용하지 않고, 서울 평균 비교는 확보 가능한 다른 공개 API 또는 보유 패널 내 강서구 행정동 비교부터 수행한다.
