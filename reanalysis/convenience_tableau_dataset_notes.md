## 3. Tableau 입력 컬럼 정의

`tableau_convenience_stores_gangseo.csv`는 Tableau에서 ‘텍스트 파일’ 또는 ‘CSV’로 바로 연결할 수 있다. Tableau에서 `longitude`를 경도, `latitude`를 위도로 지정하면 강서구 편의점 점 지도를 만들 수 있다.

| 컬럼명 | 의미 | Tableau 사용 방식 |
|---|---|---|
| `candidate_store_id` | 분석용 편의점 후보 ID | 점포별 고유 식별자, 상세 테이블의 키 |
| `source_store_id` | 원자료 상가업소번호 | 원자료 추적용 키 |
| `store_name` | 상호명 | 지도 라벨, 상세 tooltip |
| `brand` | 브랜드 정규화 값 | 색상 구분, 브랜드 필터 |
| `admin_dong` | 행정동 | 행정동별 집계, 필터 |
| `legal_dong` | 법정동 | 세부 주소 맥락 |
| `road_address` | 도로명주소 | 최종 후보 검증 및 지도 검색 |
| `jibun_address` | 지번주소 | 건물·현장 확인 보조 |
| `longitude` | 경도 | Tableau 지도 X 좌표 |
| `latitude` | 위도 | Tableau 지도 Y 좌표 |
| `biz_l1`~`biz_l3` | 업종 분류 | 원자료 추출 근거 |
| `ksic_name` | 표준산업분류명 | 원자료 추출 근거 |
